# Editing the Graph View

1. First, you will see this graph view
   ![Pasted image 20260106134136.png](./pics/Pasted%20image%2020260106134136.png)
2. Go to Graph settings
   ![Pasted image 20260106134242.png](./pics/Pasted%20image%2020260106134242.png)
3. In the Display menu, change Node size to 2.5, Link thickness to 5, and enable Arrows
   ![Pasted image 20260106134415.png](./pics/Pasted%20image%2020260106134415.png)
4. In the Forces menu, change Link distance to 100
   ![Pasted image 20260106134743.png](./pics/Pasted%20image%2020260106134743.png)
5. In the Groups menu, add colors with the following settings
   ```
   path:peoplesparty-policy/A_ปฏิรูปรัฐและระบบราชการ  
   
   ```
   ```
   path:peoplesparty-policy/B_ประชาธิปไตยและความมั่นคงใหม่  
   ```
   ```
   path:peoplesparty-policy/C_คุณภาพชีวิต  
   ```
   ```
   path:peoplesparty-policy/D_โมเดลเศรษฐกิจใหม่  
   ```
